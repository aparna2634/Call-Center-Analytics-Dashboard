# Call Center Analytics Dashboard

## Overview
This project focuses on analyzing call center operations using an interactive Power BI dashboard. The goal is to monitor key performance metrics such as call volume, SLA compliance, agent performance, and revenue, and to derive insights that help improve efficiency and customer service.

The dashboard is built using a dataset of over 1.14 million call records and provides a clear view of operational trends and business performance.

---

## Tech Stack Used
- **Power BI** – Dashboard creation and visualization  
- **DAX (Data Analysis Expressions)** – KPI calculations and measures  
- **Data Modeling** – Structuring relationships between data  

---

## Features
- 📞 **Call Volume Analysis** – Tracks total calls and trends over time  
- ⏱ **SLA Monitoring** – Measures calls within and outside SLA  
- 👨‍💼 **Agent & Manager Performance** – Evaluates efficiency and productivity  
- 📊 **Time-Based Insights** – Identifies peak hours and wait times  
- 📍 **Location Analysis** – Compares performance across different sites  
- 💰 **Revenue Insights** – Analyzes revenue generated and loss due to abandoned calls  
- 🔍 **Interactive Filters** – Enables drill-down and dynamic analysis  

---

## Conclusion
This dashboard helps in identifying key operational issues such as SLA breaches and call abandonment, which impact customer experience and revenue. By using these insights, organizations can optimize resource allocation, reduce wait times, and improve overall service efficiency.
